GNU RISC-V Embedded GCC is the RISC-V cross-compiler variant of
the [GNU Compiler Collection](https://gcc.gnu.org),
an open source project maintained by the
[Free Software Foundation](https://www.fsf.org).

:::info

In this context, _**embedded**_ refers exclusively to **bare-metal** systems,
not embedded Linux, which necessitates a different toolchain for building
Linux user space applications.

:::
